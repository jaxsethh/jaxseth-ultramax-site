# JaxSeth Ultramax Site

A premium personal brand website for JaxSeth built with Next.js App Router, TypeScript, Tailwind CSS, Motion, and Lucide icons.

## Run locally

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000

## Deploy

Push to GitHub, import the repository in Vercel, add environment variables, then deploy.
